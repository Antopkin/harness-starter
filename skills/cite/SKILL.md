# cite → digest

Этот навык влит в `digest`. Оформление библиографических ссылок — запись BibTeX и
готовая строка в стиле APA 7 / ГОСТ Р 7.0.100–2018 из метаданных работы — теперь
делает `digest`: он выдаёт оформленную ссылку финальным блоком конспекта
(`## Ссылка для списка литературы` + `## Цитирование (.bib)`), а нужна только
ссылка без конспекта — тот же финальный блок используется отдельно.

Протокол и справочник здесь не дублируются — единственный источник правды:
`skills/digest/SKILL.md` (финал «оформление ссылки и BibTeX») и
`skills/digest/references/bibtex-fields.md` (типы записей, обязательные поля,
примеры APA 7 и ГОСТ).
